
1. Створити компоненти AllUsers, Info, Default
   (Info, Default - заповнити довільно).

2. В компоненті AllUsers - відобразити список юзерів.

3. При кліку на юзера через useParams і динамічні url передавати
   id юзера в компоненту User

4. При переході на деталі юзера показати юзера поля
   (для перехоів юзати useNavigate).
